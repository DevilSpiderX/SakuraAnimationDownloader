# 使用

在动漫链接的输入框中输入动漫链接，即播放视频的[链接](http://www.imomoe.la/player/8006-0-0.html)，
然后点击查询就可以了

# 配置说明

配置文件为当前目录下的`config.json`，第一次运行会自动生成配置文件
- `DOWNLOAD_DIR` **下载保存路径** 默认为当前目录
- `TEMP_DIR` **缓存路径** 默认为当前目录
- `PROXY` **代理地址** 默认为空，格式：http://127.0.0.1:10809 或 socks5://127.0.0.1:10808
- `MAX_NUM_PARALLEL_DOWNLOAD` **最大并发下载数** 默认为3
- `RECORDING_PATH` **历史记录文件路径** 默认为当前目录
- `LOG_DIR` **日志路径** 默认为当前目录
- `MAX_LOG` **最大日志数量** 默认为7
